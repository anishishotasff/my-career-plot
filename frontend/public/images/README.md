# Profile Images

## How to Add Profile Pictures

1. Save your profile photos in this directory
2. Recommended format: JPG or PNG
3. Recommended size: 400x400px (square)
4. File names:
   - `anish.jpg` - Founder photo
   - `pilat.jpg` - Co-founder photo

## Image Guidelines:

- Use high-quality images
- Square aspect ratio works best
- File size: Keep under 500KB for fast loading
- Supported formats: JPG, PNG, WebP
